# Changelog

## 0.2.0

- GitHub release workflow and release documentation added.
- Windows controller source hardened for local/private network requests.
- Windows controller serializes profile changes to avoid overlapping monitor commands.
- macOS build moved to a real Xcode application project generated by XcodeGen.
- macOS discovery keeps controller identity stable while its IP changes.
- G915 monitor uses IOBluetooth first and preserves the proven system-profiler fallback.
- Generic setup documentation now avoids any user-specific IP, token or monitor name.
